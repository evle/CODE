# Dockerfile

## docker文件流程



[[readme]]

为容器提供默认的执行命令, 通过 `CMD` 和 `ENTRYPOINT`, 必须指定一个, 否则会报错

exec 写法

```
CMD ["exec", "param1", "param2", "$HOME"]

# 不执行shell所以$HOME读取不到, 使用sh才行

CMD [ "sh", "-c", "echo $HOME" ]
```

shell 写法:

```
CMD exec param1 param2
```
